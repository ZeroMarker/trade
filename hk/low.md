## 什么是低门槛港股

**低门槛 = 入场费低（股价低 × 每手股数少）**

港股每手股数由上市公司自行设定，从 100 股到 10,000 股不等。真正低门槛的股票需要同时满足：

- **股价低**（通常 < 1 HKD）
- **每手股数少**（1000 股或更少）
- **入场费 < 200 HKD**

利用汇丰 Trade25 计划（买入佣金 25 HKD 封顶），低门槛策略才能成立。否则佣金占比过高会严重侵蚀收益。

## 费用结构分析

### 使用 Trade25 的成本模型

| 入场费 | Trade25 佣金 | 其他费用 | 总成本 | 佣金占比 |
|--------|-------------|---------|--------|---------|
| 50 HKD | 25 HKD | ~10 HKD | ~85 HKD | 50% |
| 200 HKD | 25 HKD | ~10 HKD | ~235 HKD | 12.5% |
| 500 HKD | 25 HKD | ~10 HKD | ~535 HKD | 5% |
| 1000 HKD | 25 HKD | ~10 HKD | ~1035 HKD | 2.5% |

**关键结论：** 入场费低于 200 HKD 时，交易成本占比极高。只适合**练手**或**极小仓位试探**，不适合频繁交易。

### 不使用 Trade25 的情况

标准港股佣金通常为成交额的 0.25% + 最低收费（~100 HKD），外加印花税、交易征费、结算费等。

**低门槛交易必须依赖 Trade25 或同类免佣/低佣计划。**

## 高风险机会型

适用于投机心态，对亏损有充分准备。

### 1. 国美零售 (00493)

| 项目 | 内容 |
|------|------|
| 入场费 | ~14 HKD（历史低位） |
| 每手股数 | 1,000 |
| 风险等级 | ⚠️⚠️⚠️⚠️⚠️ |
| 类型 | 仙股 |

**背景：** 国美零售近年陷入严重财务困境，经历重组。股价极低，接近"仙股"水平。

**核心逻辑：**
- 纯投机：押注重组成功或利好带来的反弹
- 入场费极低，即使全亏也在可接受范围
- 但从基本面看持续亏损，现金消耗严重

**建议策略：**
- 仅以"彩票心态"投入极小金额
- 设置 -20% 止损线
- 关注重组进展公告

### 2. 新火集团 (01611)

| 项目 | 内容 |
|------|------|
| 入场费 | ~30-100 HKD |
| 每手股数 | 视调整而定 |
| 风险等级 | ⚠️⚠️⚠️⚠️ |
| 类型 | 概念股 |

**背景：** 与虚拟资产/区块链概念相关，有名人效应加持。

**核心逻辑：**
- 纯概念驱动，波动剧烈
- 虚拟资产市场情绪对其股价影响较大
- 基本面不稳定，不适合重仓

**建议策略：**
- 仅作为虚拟资产市场情绪的"温度计"
- 小仓位快进快出
- 与 BTC 价格走势对照分析

### 3. 中国金融国际 (00721)

| 项目 | 内容 |
|------|------|
| 入场费 | ~20-50 HKD |
| 风险等级 | ⚠️⚠️⚠️⚠️⚠️ |
| 类型 | 仙股 |

**背景：** 长期低价仙股，流动性极差。

**核心逻辑：**
- 极高风险，需注意流动性问题（可能无法按预期价格卖出）
- 不适合资金量较大的仓位
- 只适合极小资金试探

**建议策略：**
- 仅适合了解仙股交易机制
- 严格限制仓位
- 重视流动性风险

## 低估修复型

适合基本面投资者，风险相对可控。

### 4. 翼辰实业 (01596)

| 项目 | 内容 |
|------|------|
| 入场费 | ~100-300 HKD（视价格波动） |
| 风险等级 | ⚠️⚠️⚠️ |
| 类型 | 基本面反转 |

**背景：** 公司主营业务涉及铁路相关设备，2025 年净利润实现扭亏。

**核心逻辑：**
- 业绩已出现拐点——从亏损转为盈利
- 属于工业/基建板块，与国策（铁路基建）相关
- 但需警惕：扭亏后股价可能已经充分反映利好

**关键指标关注：**
- 扭亏是否可持续（非经常性损益 vs 核心业务改善）
- 营收是否同步增长
- 现金流状况
- PB 是否仍处于低位

**建议策略：**
- 相比仙股更适合基本面投资
- 入场费仍在可接受范围内
- 关注后续财报验证业绩持续性
- 设立 -15% 止损

## 风险控制

### 低门槛交易的核心原则

| 原则 | 说明 |
|------|------|
| 仓位限制 | 单只低门槛股票不超过总资金的 5% |
| 总敞口限制 | 所有低门槛投机仓位不超过总资金的 15% |
| 止损纪律 | 每笔交易预设止损点（推荐 -10%~-20%） |
| 流动性检查 | 交易前检查过去 30 天日均成交量 |
| 时间止损 | 持有 1 个月无起色，考虑退出 |

### 仙股特别风险

1. **流动性枯竭：** 可能连续多日无成交，无法卖出
2. **缩股：** 公司可能进行缩股（合股），导致名义股价看似上涨但实际价值不变
3. **停牌/除牌：** 长期停牌导致资金被锁定
4. **大比例稀释：** 公司可能通过配股/供股大幅稀释小股东权益

### 适合低门槛的场景

- **初入港股：** 用极低成本了解港股交易规则、操作流程
- **验证策略：** 用小额资金验证某个思路或假设
- **情绪感知：** 通过低价股感受市场情绪变化
- **特定事件驱动：** 押注某个重组/公告 events

### 不适合低门槛的场景

- **定投：** 佣金占比过高，侵蚀复利效果
- **高频交易：** 每次 25 HKD 的佣金对极低金额来说占比过高
- **大资金分批建仓：** 更好的选择是直接买入正股或 ETF

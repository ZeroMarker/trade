# 汇丰 Trade25 计划

## 什么是 Trade25

汇丰 Trade25 是香港上海汇丰银行推出的一项港股交易佣金优惠计划。核心卖点是 **买入佣金上限 25 HKD**，无论成交金额大小。

### 基本规则

| 项目 | 内容 |
|------|------|
| 买入佣金 | **每笔交易上限 25 HKD**（无论成交金额） |
| 卖出佣金 | 按标准费率或促销费率（通常 ~0.25%） |
| 适用市场 | 香港联交所主板及创业板 |
| 适用账户 | 汇丰综合投资账户 / 股票账户 |
| 月费 | 通常无额外月费（视账户等级） |

---

## 核心优势

### 1. 低门槛交易成为可能

没有 Trade25，买入 100 HKD 的港股按标准佣金（0.25% 最低 ~100 HKD），光佣金就超过交易金额本身。

有了 Trade25，小额交易的佣金成本被锁定在 25 HKD，使得以下场景变得可行：

- 买入 100 HKD 的股票 → 佣金 25 HKD（占比 25%，仍偏高但可接受）
- 买入 500 HKD 的股票 → 佣金 25 HKD（占比 5%，合理）

### 2. 适合的策略

| 策略 | 适合度 | 说明 |
|------|--------|------|
| 小额定投 | ✅ | 每月固定小额买入特定标的 |
| 试探建仓 | ✅ | 先用小仓位测试策略再追加 |
| 练手学习 | ✅ | 低成本熟悉港股交易流程 |
| 短线交易 | ⚠️ | 买入成本低，但卖出成本仍高 |
| 大额定投 | ❌ | 大额投资用标准费率更划算 |

## 费用详解

### Trade25 交易总成本组成

以买入 200 HKD 的股票为例：

| 费用项目 | 金额 | 说明 |
|---------|------|------|
| 佣金（Trade25） | 25 HKD | 汇丰收取 |
| 印花税 | 0.2 HKD | 成交金额 × 0.13%（政府，每千元向上取整至元） |
| 交易征费 | ~0.06 HKD | 成交金额 × 0.0027%（证监会） |
| 交易费 | ~0.01 HKD | 成交金额 × 0.00565%（联交所） |
| 结算费 | ~0.02 HKD | 成交金额 × 0.005%（最高 200 HKD） |
| **买入总成本** | **~25.30 HKD** | |

### 标准费率 vs Trade25 对比

| 成交金额 | 标准佣金（~0.25%） | Trade25 | 节省 |
|---------|-------------------|---------|------|
| 1,000 HKD | ~100 HKD（最低收费） | 25 HKD | 75% |
| 5,000 HKD | ~100 HKD（最低收费） | 25 HKD | 75% |
| 10,000 HKD | 25 HKD | 25 HKD | 持平 |
| 50,000 HKD | 125 HKD | 25 HKD | 80% |
| 100,000 HKD | 250 HKD | 25 HKD | 90% |

**关键分界点：** 成交金额在 10,000 HKD 左右时，标准费率和 Trade25 成本相当。低于此金额时 Trade25 更优，高于此金额时标准费率可能更优。

> 注：以上标准佣金假设 0.25% 费率 + 最低收费 100 HKD，不同券商/账户等级可能不同。

## 局限性

### 1. 买低卖高两头吃

Trade25 只优惠**买入**端，**卖出**仍按标准费率。这意味着：

- 买入成本：锁定 25 HKD ✅
- 卖出成本：按成交金额比例收费 ❌
- **需要盈利更多才能覆盖总成本**

### 2. 小额交易的隐形成本占比高

即使佣金被封顶，其他固定费用占比在小额交易中仍然显著：

- 印花税：成交额 × 0.13%
- 交易征费 + 交易费 + 结算费：合计 ~0.014%

但好在这部分费用绝对值很低（手续费 < 1 HKD），对总成本影响有限。

### 3. 卖出时机受限

由于卖出按标准费率，Trade25 用户自然的倾向是：

- **买入分散**：分多次小额买入（每次成本 25 HKD）✅
- **集中卖出**：累积到一定仓位后一次性卖出 ❌（大额卖出佣金高）

更好的做法是：Trade25 买入积累仓位，卖出时视情况分批出场，或接受一次性卖出的佣金成本。

## 适合与不适合的场景

### 适合

- **新手入门：** 用几百 HKD 开始港股投资，学习操作流程
- **策略试验：** 以低成本验证投资策略的有效性
- **仙股/低价股：** 14 HKD 的国美零售买 10 手，总成本也不到 200 HKD
- **碎片化投资：** 利用散碎资金入场

### 不适合

- **大额交易：** 10 万以上的买入，25 HKD 封顶仍然便宜，但不如使用固定佣金计划划算
- **频繁交易（高频）：** Trade25 不是为日内高频交易设计的
- **期权/衍生品：** 不适用，Trade25 仅限正股和 ETF

## 与同类方案对比

| 方案 | 买入佣金 | 卖出佣金 | 月费 | 适合 |
|-----|---------|---------|------|------|
| 汇丰 Trade25 | 封顶 25 HKD | 标准费率 | 无 | 小额买入 |
| 银行股无限次买入 | 0 佣金 | 标准费率 | 可能有 | 高频定投 |
| 互联网券商（富途/老虎） | 0~5 HKD | 0~5 HKD | 无 | 全场景 |
| 传统银行标准计划 | 最低 100 HKD | 最低 100 HKD | 无 | 大额交易 |

### 互联网券商对比

Trade25 的存在意义在不同时期不同：

- **早年（无互联网券商时代）：** Trade25 是低门槛交易的唯一选择
- **当前（富途/老虎等 0 佣金时代）：** Trade25 的优势被削弱，但仍有用武之地
  - 如果你已有汇丰账户，额外开 Trade25 无缝衔接
  - 对于不方便开互联网券商账户的用户（如某些地区限制）

---

## 总结：Trade25 的定位

```
Trade25 不是最佳方案，而是"方案组合"中的一环：

- 小额试探 / 练手 → Trade25
- 日常定投 / 小额建仓 → Trade25  
- 大额买入 / 中长期持仓 → 标准费率（更低的有效费率）
- 高频交易 → 互联网券商 0 佣金
```

> 注：以上信息基于公开资料整理，具体费率以汇丰银行最新公告为准。费率结构和规则可能随时调整。

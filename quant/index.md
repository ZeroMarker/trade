## 什么是量化交易

量化交易是用数学模型和计算机算法来做出交易决策，而非依赖主观判断。

### 核心流程

```
思路 → 假设 → 回测 → 优化 → 模拟 → 实盘 → 监控
```

| 步骤 | 说明 | 工具/方法 |
|------|------|----------|
| 思路 | 发现市场规律或逻辑 | 观察、研报、论文 |
| 假设 | 将思路转化为可验证的规则 | 数学表达 |
| 回测 | 用历史数据验证策略表现 | Python、回测框架 |
| 优化 | 调整参数、控制过拟合 | 参数扫描、交叉验证 |
| 模拟 | 在实时市场模拟交易 | Paper Trading |
| 实盘 | 投入真金白银执行 | 自动化交易系统 |
| 监控 | 持续评估策略表现 | 绩效报告、风控 |

### 适合量化的市场特性

| 市场 | 适合度 | 说明 |
|------|--------|------|
| 美股 | ✅✅ | 流动性好、数据全、支持算法交易（IBKR API） |
| 港股 | ✅ | 流动性分化、数据不如美股全 |
| A 股 | ✅ | 流动性好但 T+1 + 涨跌停限制大 |
| 加密货币 | ✅✅ | 7×24 小时、波动大、数据开放 |

## 策略类型总览

| 策略类型 | 逻辑 | 持有期 | 年化目标 | 最大回撤 |
|---------|------|--------|---------|---------|
| 趋势跟踪 | 追涨杀跌 | 数天~数月 | 10-20% | 15-30% |
| 均值回归 | 高抛低吸 | 数天~数周 | 8-15% | 10-20% |
| 因子策略 | 系统性因子暴露 | 数月~数年 | 8-15% | 15-25% |
| 网格交易 | 震荡市高抛低吸 | 分钟~天数 | 5-15% | 5-10% |
| 统计套利 | 配对交易价差回归 | 分钟~天数 | 8-12% | 5-10% |
| 高频交易 | 微秒级订单流博弈 | 毫秒~秒 | 不定 | 极低 |

## 趋势跟踪

### 核心逻辑

"趋势是你的朋友"——在趋势确立后入场，在趋势反转时退出。

### 双均线交叉 (MA Crossover)

| 参数 | 说明 |
|------|------|
| 逻辑 | 快线上穿慢线 → 买入；快线下穿慢线 → 卖出 |
| 常用参数 | 快线 5/10/20 日，慢线 50/100/200 日 |
| 适用市场 | 趋势明显的市场（美股、BTC） |
| 局限性 | 震荡市中反复打脸 |

### 海龟交易法则 (Turtle Trading)

| 参数 | 说明 |
|------|------|
| 入场 | 突破过去 N 日高点 → 做多 |
| 止损 | 跌破过去 M 日低点 → 止损 |
| 加仓 | 每涨 0.5 ATR 加仓一次 |
| 经典参数 | N=20（入场），M=10（止损） |

### 趋势跟踪的评价

| 优势 | 劣势 |
|------|------|
| 牛市能吃到完整涨幅 | 震荡市亏损 |
| 逻辑简单、容易理解 | 磨损严重（频繁假信号） |
| 适用标的广 | 连续亏损期考验心态 |

## 均值回归

### 核心逻辑

"涨多了会跌，跌多了会涨"——价格围绕均值波动，偏离越大回归概率越高。

### RSI 反转

| 参数 | 说明 |
|------|------|
| 逻辑 | RSI < 30 → 超卖，买入；RSI > 70 → 超买，卖出 |
| 常用参数 | 周期 14，超卖 30，超买 70 |
| 适用 | 震荡市、横盘标的 |
| 不适用 | 强趋势市场（会过早止盈/止损） |

### 布林带反转 (Bollinger Bands)

```
中轨 = SMA(20)
上轨 = SMA(20) + 2 × 标准差
下轨 = SMA(20) - 2 × 标准差

触及下轨 → 买入（预期回归中轨）
触及上轨 → 卖出（预期回归中轨）
```

## 因子策略

### 经典因子

| 因子 | 逻辑 | A 股表现 | 美股表现 |
|------|------|---------|---------|
| 价值 (Value) | 低 PE/PB 跑赢 | ⚠️ 价值陷阱风险 | ✅ 长期有效 |
| 动量 (Momentum) | 强者恒强 | ❌ 反转效应更强 | ✅ 有效 |
| 质量 (Quality) | 高 ROE、低负债 | ✅ 有效 | ✅ 有效 |
| 低波 (Low Vol) | 低波动跑赢 | ✅ 有效 | ✅ 有效 |
| 红利 (Dividend) | 高股息跑赢 | ✅ 有效 | ✅ 有效 |
| 小市值 (Size) | 小盘跑赢大盘 | ✅ 明显有效 | ⚠️ 近十年失效 |

### 因子轮动

| 市场环境 | 占优因子 |
|---------|---------|
| 经济扩张 + 低利率 | 小市值、动量 |
| 经济扩张 + 高利率 | 价值、红利 |
| 经济收缩 + 低利率 | 质量、低波 |
| 经济收缩 + 高利率 | 红利、价值 |

## 网格交易

### 原理

在震荡市中预设价格网格，低买高卖完全自动化。

```
网格设置（当前价 100）：
  110 → 卖出 100 股
  105 → 卖出 100 股
  100 → 底仓
  95  → 买入 100 股
  90  → 买入 100 股
```

### 参数设计

| 参数 | 说明 | 经验值 |
|------|------|--------|
| 网格数量 | 分几层 | 5-10 层 |
| 网格间距 | 格子之间的距离 | 1-5% |
| 上下边界 | 网格范围 | 当前价 ± 10-20% |

### 风险

- **单边行情：** 价格突破网格范围 → 空仓或满仓
- **趋势市表现差：** 牛市中早早卖光
- **资金利用率低：** 部分资金始终空闲

## 统计套利

### 配对交易 (Pairs Trading)

#### 步骤

1. 寻找高度相关的两个标的（如 SPY & IVV）
2. 计算价差的均值与标准差（Z-Score）
3. 价差偏离过大时开仓（做多弱的、做空强的）
4. 价差回归时平仓

```
Z-Score = (当前价差 - 均值价差) / 标准差

Z-Score > +2 → 做空价差（卖强买弱）
Z-Score < -2 → 做多价差（买强卖弱）
```

| 优势 | 劣势 |
|------|------|
| 市场中性（对冲大盘风险） | 需要做空能力 |
| 回撤较小 | 需要大资金 |
| 逻辑清晰 | 协整关系可能破裂 |

## 回测框架

### 常用工具

| 框架 | 语言 | 特点 | 适合 |
|------|------|------|------|
| Backtrader | Python | 功能完善、社区大 | 个人投资者 |
| Zipline | Python | Quantopian 出品 | 因子研究 |
| VectorBT | Python | 极速向量化回测 | 参数扫描 |
| 聚宽 (JoinQuant) | Python | 在线+A 股数据 | A 股用户 |
| TradingView | Pine Script | 在线、可视化好 | 趋势策略验证 |
| QuantConnect | Python/C# | 云端回测、数据全 | 多市场、多资产 |

### 核心回测指标

| 指标 | 说明 | 参考范围 |
|------|------|---------|
| 年化收益率 | 策略的年化复合回报 | > 10-15% |
| 夏普比率 | (收益 - 无风险) / 波动率 | > 1.0（好），> 2.0（极好） |
| 最大回撤 | 从峰值到谷底的最大跌幅 | < 20% |
| 卡玛比率 | 年化收益 / 最大回撤 | > 1.0 |
| 胜率 | 盈利次数 / 总次数 | 视策略而定 |
| 盈亏比 | 平均盈利 / 平均亏损 | > 1.5 |

## 常见误区

### 过拟合 (Overfitting)

| 现象 | 说明 |
|------|------|
| 回测完美、实盘惨淡 | 参数过度优化，拟合了历史噪音 |
| 大量规则、复杂逻辑 | 参数越多越容易过拟合 |

**避免方法：**
1. 保持策略简单（奥卡姆剃刀）
2. 样本外测试不用时间段
3. 参数敏感性分析
4. 蒙特卡洛模拟

### 陷阱检查清单

| 陷阱 | 说明 |
|------|------|
| 前视偏差 (Look-ahead Bias) | 回测中用了未来数据 |
| 幸存者偏差 | 只考虑现存标的，忽略退市的 |
| 交易成本忽略 | 未计入佣金、滑点、冲击成本 |

### 实盘与回测的差距

```
回测年化 20% → 实盘年化 10-15%（含滑点、交易成本）
回测夏普 2.0 → 实盘夏普 1.0-1.5（策略衰减）
回测最大回撤 10% → 实盘最大回撤 15-20%（极端行情）
```

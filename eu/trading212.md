## 简介

Trading 212 是一家总部位于英国的金融科技券商，2013 年成立。核心定位是**零佣金零售券商**，与 Robinhood 模式类似。

| 项目 | 内容 |
|------|------|
| 总部 | 英国伦敦 |
| 监管 | FCA（英国）、CySEC（塞浦路斯） |
| 成立 | 2013 年 |
| 用户数 | ~500 万+（欧洲地区） |
| 核心卖点 | 零佣金、碎股、免手续费 |
| 语言 | 支持中文（界面部分翻译） |

### 与 IBKR 的定位差异

| 维度 | Trading 212 | IBKR |
|------|-------------|------|
| 目标用户 | 散户/小白投资者 | 严肃交易者/专业投资者 |
| 界面 | 简洁、移动端优先 | 复杂（TWS 学习曲线陡峭） |
| 佣金 | 零佣金 | 低佣金（$0.35 起） |
| 碎股 | ✅ 支持 | ✅ 支持 |
| 全球市场 | 有限（主要美股+英股+欧股） | 150+ 市场 |
| 量化/API | ❌ 不支持 | ✅ TWS API |
| 融资利率 | 较高 | 行业最低 |

## 账户类型

### 投资账户 (Invest Account)

| 属性 | 内容 |
|------|------|
| 交易标的 | 股票、ETF |
| 杠杆 | 无 |
| 佣金 | 免费 |
| 适合 | 长期现货投资 |

### CFD 账户 (CFD Account)

| 属性 | 内容 |
|------|------|
| 交易标的 | CFD（差价合约）— 股票、指数、商品、外汇、加密货币 |
| 杠杆 | 有（最高 1:30，FCA 限制） |
| 佣金 | 免费（但有点差） |
| 适合 | 短线交易、做空 |

### ISA 账户 (英国)

| 属性 | 内容 |
|------|------|
| 适用人群 | 英国税务居民 |
| 免税额度 | £20,000/年 |
| 资本利得税 | 0%（ISA 内） |
| 股息税 | 0%（ISA 内） |

## 费用结构

### 股票/ETF 交易

| 费用项 | Invest Account | CFD Account |
|--------|---------------|-------------|
| 佣金 | **0** | **0** |
| 托管费 | 0 | 0 |
| 账户管理费 | 0 | 0 |
| 货币转换费 | 0.15%（用外币交易时） | 包含在点差中 |
| 不活跃费 | 0 | 0 |
| 存款费 | 0 | 0 |

### 点差成本

实际交易成本隐藏在点差中。Trading 212 通过"加宽点差"来获得收入（Payment for Order Flow, PFOF 模式）。

| 资产类别 | 典型点差 | 对比 IBKR |
|---------|---------|-----------|
| 主要美股（苹果、SPY 等） | ~0.05% | 略高于 IBKR |
| 小盘美股 | ~0.1-0.3% | 差距更大 |
| 主要 ETF | ~0.03% | 接近 |
| 外汇（主要货币对） | ~1 pip | 高于 IBKR |

> **注意：** 零佣金不代表零成本。Trading 212 通过订单流付费 (PFOF) 和加宽点差来盈利。这意味着大额交易的隐性成本可能超过 IBKR 的直接佣金。

## 交易美股

### 碎股交易

Trading 212 支持碎股（Fractional Shares），是其核心卖点之一：

- **最小投资额：** $1
- **适合定投：** 每月固定金额买入指定股票/ETF
- **股息分配：** 按持股比例分配

### 定投功能 (AutoInvest)

Trading 212 的 AutoInvest 是其差异化优势：

1. 设置每月/每周定投计划
2. 选择标的和金额
3. 系统自动执行（支持碎股）

**适合的定投标的：**

| 标的 | 定投理由 |
|------|---------|
| VOO / VUSA | 标普 500 指数 |
| VWCE / VWRA | 全球指数 |
| QQQ / EQQQ | 纳斯达克 100 |
| AAPL / MSFT | 蓝筹科技股 |

### 美股交易细则

- **交易时间：** 常规 9:30-16:00 ET
- **盘前盘后：** Trading 212 支持有限（不如 IBKR 全面）
- **结算：** T+2（欧盟规则）

## 交易港股

### Trading 212 港股支持

- **标的数量：** 有限（仅支持部分主要港股，如腾讯、阿里、友邦等）
- **覆盖面：** 远不如 IBKR 和汇丰
- **碎股：** 不支持港股碎股

### 港股交易注意事项

| 项目 | 内容 |
|------|------|
| 标的选择 | 仅有恒生大蓝筹（腾讯 00700、阿里 09988、友邦 01299 等） |
| 小盘股/仙股 | ❌ 不支持 |
| 费用 | 零佣金，但有货币转换费（GBP/HKD/USD 之间） |
| 流动性 | 不如直接在港股券商交易 |

**结论：** Trading 212 **不是交易港股的好选择**。港股还是汇丰或 IBKR 更优。

## 交易欧股/英股

Trading 212 在欧股/英股方面是强项（毕竟是欧洲券商）：

| 市场 | 佣金 | 支持标的 | 说明 |
|------|------|---------|------|
| 英国 LSE | 免费 | 多数 FTSE 标的 | 无需货币转换（GBP） |
| 德国 Xetra | 免费 | DAX 主要标的 | 欧元计价 |
| 法国 Euronext | 免费 | CAC 40 标的 | 欧元计价 |
| 其他欧洲 | 免费 | 主要标的 | 视具体市场 |

**与 IBKR 对比：**

- **Trading 212：** 零佣金，标的覆盖主要欧洲蓝筹
- **IBKR：** 低佣金（~3 GBP），标的覆盖更全，包括小盘股

## CFD 与差价合约

### CFD 费用模型

| 项目 | 内容 |
|------|------|
| 点差 | 主要货币对 ~1 pip，指数 ~1-2 点 |
| 隔夜融资费 | 有（多空不同方向费率不同） |
| 佣金 | 零（股票 CFD 也没有佣金） |
| 杠杆 | 最高 1:30（FCA 零售客户） |

### 杠杆限制（FCA）

- 加密货币 CFD：1:2
- 股票 CFD：1:5
- 外汇主要货币对：1:30
- 指数：1:20

### CFD 风险警示

> Trading 212 数据显示，**约 74-82% 的 CFD 散户投资者亏损**。CFD 是复杂产品且带杠杆，可能导致快速亏损。

## 税务要点

### 非英国居民

- **资本利得税：** 无需缴纳英国 CGT
- **股息税：** UK 来源股息需缴税，非 UK 来源股息不用
- **W-8BEN：** 美股股息预扣税 15%（需提交 W-8BEN）

### 英国居民

- **ISA 账户：** 每年 £20,000 额度，免税增长
- **一般投资账户：** CGT 免税额 £6,000（2023/24 年度），超出部分按 10-20% 征税
- **Dividend Allowance：** 股息收入前 £1,000 免税

## IBKR 对比总结

### 核心场景判断

| 场景 | 推荐 | 原因 |
|------|------|------|
| 欧洲居民、长期定投 | **Trading 212** | 零佣金、AutoInvest、碎股 |
| 全球市场配置 | **IBKR** | 150+ 市场、港股+A 股 |
| 小额交易 | **Trading 212** | 零佣金不怕成本占比高 |
| 大额交易 | **IBKR** | 点差更低，真实成本更低 |
| 量化/API | **IBKR** | Trading 212 不支持 |
| 港股/仙股 | **IBKR / 汇丰** | Trading 212 港股覆盖极有限 |
| 期权/期货 | **IBKR** | Trading 212 不支持 |
| 简简单单买美股 | **Trading 212** | 零成本、体验好 |

### 一句话总结

```
Trading 212 = 欧洲版 Robinhood。适合零佣金长期定投美股 ETF，体验极好。
IBKR = 专业工具。适合全球配置、多市场、量化交易，成本真实更低。

两者不冲突，可以同时使用：
- Trading 212 用于定投 ETF（自动、免费、省心）
- IBKR 用于主动交易、港股、期权、全球市场
```

> 注：以上信息基于公开资料整理，具体费率和规则以 Trading 212 官方最新公告为准。
